# reprizafetch

just a fetch by repriza

## Dependencies

**Arch-based** (CachyOS, Manjaro, etc.)
```bash
sudo pacman -S python
```

**Debian-based** (Ubuntu, Mint, etc.)
```bash
sudo apt install python3
```

**Fedora-based** (Nobara, etc.)
```bash
sudo dnf install python3
```

## Install
```bash
git clone https://github.com/reprizawalker/reprizafetch.git
cd reprizafetch
chmod +x reprizafetch.py
sudo cp $(pwd)/reprizafetch.py /usr/local/bin/reprizafetch
```
```

## Usage
```bash
reprizafetch
```
